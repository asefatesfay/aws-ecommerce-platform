# Divide and Conquer

Divide and conquer breaks a problem into smaller subproblems, solves each independently, and combines the results. It's the foundation of merge sort, quicksort, and many tree algorithms.

## Core Pattern

1. **Divide:** Split the problem into smaller subproblems
2. **Conquer:** Solve each subproblem recursively (base case when small enough)
3. **Combine:** Merge the solutions

## When to Use

- Problem can be split into independent subproblems
- Subproblems are the same type as the original
- Combining solutions is efficient
- Examples: sorting, binary search, tree construction, closest pair of points
